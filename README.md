## Usage

<pre>npm i</pre>

use Mysql to login and access DB
then run:

<pre>SOURCE db\schema.sql</pre>
<pre>exit</pre>

Then to seed the db, run:
<pre>node run seed</pre>

To run the server, run:

<pre>node server.js</pre>
