## Usage

<pre>npm i</pre>

Using a local database, ecommerce_db,

<pre>mysql -u root -p</pre>

Enter mysql db info

<pre>SOURCE db\schema.sql</pre>
<pre>exit</pre>

Then seed the db, run:

<pre>node run seed</pre>

To run the server, run:

<pre>node server.js</pre>
